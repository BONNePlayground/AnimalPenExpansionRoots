# Animal Pens Expansion - Roots

This mod is a compatibility addon that allows mobs from Roots to work with the pens from Animal Pens, making animal management easier and more consistent.

## Features

Animal Cage can pick up:
- Beetle - can be fed with beetle food items
- Deer - can be fed with deer food items
- Duck - can be fed with duck food items
- Fennec - can be fed with fennec food items
- Jerboa - can be fed with jerboa food items
- Green Sprout - can be fed with sprout food items which drops Sprout Gift
- Melody Sprout - can be fed with sprout food items which drops Sprout Gift
- Purple Sprout - can be fed with sprout food items which drops Sprout Gift
- Red Sprout - can be fed with sprout food items which drops Sprout Gift
- Snow Sprout - can be fed with sprout food items which drops Sprout Gift
- Tan Sprout - can be fed with sprout food items which drops Sprout Gift

Bird Catcher Pickable:
- Owl - can be fed with owl food items

## Sides:

Server side adds all interactions. 

Can be used as datapack only, however, it will be missing sprout gifts after feeding.

## 🛠️ Requirements:

📌 **[Animal Pen](https://modrinth.com/mod/animalpens)** (Required).  
📖 **[Roots](https://www.curseforge.com/minecraft/mc-mods/roots)** (Required).
